# Inventario_GCM

Proyecto base creado con Vite y Tailwind CSS.

Instalación y uso:

```bash
npm install
npm run dev
```

Inicializar git y añadir remoto (si no se hizo automáticamente):

```bash
git init
git branch -M main
git remote add origin https://github.com/itsxMasato/Inventario_GCM.git
git add .
git commit -m "Initial commit"
git push -u origin main
```
